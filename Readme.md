# AI Fitness Recommendation System

This project recommends workouts and meals using:
- Machine Learning (Random Forest)
- Rule-based filtering
- Streamlit Web App

## Features
- Predict workout intensity
- Recommend exercises based on body part
- Recommend meals based on goal
- Clean UI with custom background

## Tech Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## How to Run

```bash
streamlit run app.py
